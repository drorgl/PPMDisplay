# PPM Display

Displays Up to 8 PPM Channel Values, Shows a message when PPM Signal is unstable or disconnected.

## Config
PPM Pin 3

LCD 13 (SCLK), 11 (DIN), 5 (DC), 7 (CS), 6 (RST)
